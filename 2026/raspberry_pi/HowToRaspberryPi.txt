//==========
// ゆっくりボイス

1, 参考サイト
	https://zenn.dev/noraworld/articles/aquestalk-pi-version-1-20-ubuntu20-arm64

2, ダウンロード & インストール
	ルートフォルダにある"AquesTalkPi"は"32bitマシン用"、
	64bitフォルダの中には、"64bitマシン用"なので注意

	$ cd Desktop
	$ wget -qO- https://raw.githubusercontent.com/noraworld/aquestalk-installer/master/bin/pi | sh
	$ cd aquestalk
	$ ./AquesTalkPi これは、テストです | aplay

3, ヘルプ
	$ ./AquesTalkPi -h

4, Wavファイルに保存
	$ ./AquesTalkPi -s 150 -v f2 -k -o out.wav "ゆっくりれいむと、ゆっくりまりさだぜ" | aplay out.wav

// CRON

1, CRON起動
	$ crontab -e
	初回起動時は下記の様になり、エディタは"nano"に選択しておく

	no crontab for root - using an empty one

	Select an editor.  To change later, run 'select-editor'.
	  1. /bin/nano        <---- easiest
	  2. /usr/bin/vim.basic

	Choose 1-2 [1]: nano

2, crontabの使い方

	$ crontab -e
		# 1分おきにログを追記(とりあえずDesktopに...)
		*/1 * * * * echo "Hello, crontab!!" >> /home/<USER NAME>/Desktop/cron.log

	$ control + O
		Y -> Enterで保存

	$ control + X
		終了

	$ cat cron.log
		ログファイルが出力されていれば成功

3, crontabでpython実行

	$ crontab -e
		# ブート時に実行
		@reboot /usr/bin/python3 /home/<USER NAME>/Desktop/01/main_reboot.py
		@reboot sleep 30 /usr/bin/python3 /home/<USER NAME>/Desktop/01/main_yukkuri.py

		/usr/bin/python3 /home/kajiru/Desktop/01/main_yukkuri.py

		# 6時間毎に、リブート実行
		0 */6 * * * /usr/sbin/reboot

		# 10分おきに定期実行
		*/10 * * * * /usr/bin/python3 /home/<USER NAME>/Desktop/01/main_test.py

@daily /usr/sbin/reboot

@reboot sleep 30 && /usr/bin/python3 /home/kajiru/Desktop/01/main_yukkuri.py >> /home/kajiru/cron_error.log 2>&1

*/3 * * * * /usr/bin/python3 /home/kajiru/Desktop/01/main_test.py >> /home/kajiru/cron_error.log 2>&1

	サウンドデバイスを指定しながらテストする方法
		$ aplay -D plughw:0,0 /usr/share/sounds/alsa/Front_Center.wav
		$ aplay -D plughw:1,0 /usr/share/sounds/alsa/Front_Center.wav

	デフォルトでは、"card 0"の、HDMIに行っている。
	今回は、"card 1"の、ヘッドフォンを使うべきとわかるので、
	pythonを下記のコードに変更する。

		前: p2 = subprocess.Popen([aplay], stdin=p1.stdout)

		後: p2 = subprocess.Popen([aplay, "-D", "plughw:1,0"], stdin=p1.stdout)
